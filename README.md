## 卡牌类

### 单张卡牌

* 花色
* 点数

### 单张卡牌窗口

* 正面——显示每张扑克牌的花色和点数
* 背面——显示扑克牌背景图片
* 扑克牌所有者
* 扑克牌数据信息
* 扑克牌选中状态

###  多张卡牌

* 发牌——储存
* 出牌——删除
* 排序
* 扑克牌点数
  * 最小点数
  * 最大点数
  * 指定点数的牌数量
  * 是否包含某一张/几张牌
* 牌的数量
  * 是否为空
  * 当前数量
  * 清空

### 玩家

* 属性
  * 名字
  * 玩家分数
  * 游戏结果
    * 输
    * 赢
  * 角色
    * 农民
    * 地主
  * 性别
    * 男
    * 女
  * 头像显示位置
    * 左
    * 右
  * 玩家类型
    * 用户
    * 机器人
